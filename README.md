Как запустить эту версию 

python email_agent_local.py

Особенности локальной версии:

✅ Не требует интернета для установки дополнительных пакетов
✅ Сохраняет данные в SQLite базу (leads_database.db)
✅ Автоматически создает CSV файл (leads_export.csv) который открывается в Excel
✅ Поддерживает Gmail, Yandex, Mail.ru
✅ Извлекает телефоны, email, адреса
✅ Запоминает настройки после первого запуска
Для Gmail важно:

    Включить IMAP в настройках Gmail

    Создать пароль приложения: Настройки Google → Безопасность → Двухфакторная аутентификация → Пароли приложений
